Welcome to RetroBlog!

      This project originally started as a thought experiment. I was in the midst of an online web development boot camp, when I learned why CSS is used for syling. That got me thinking; what would it been like to be a web developer in the 1990's? Would I have even been able to make a good website?

     Enter RetroBlog: A website that uses deprecated HTML styling to create a fully functional and, at least I think, does it in a way that is somewhat aesthetically pleasing while still being useful. RetroBlog is inspired by both 1990's HTML web development and the 1960's gruvbox color scheme. We use tables for formatting, and some jenky deprecated HTML attributes for styling. Enjoy your visit, and thanks for stopping by.
