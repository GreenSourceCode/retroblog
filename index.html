      This project originally started as a thought experiment. I was in the midst of an online web development boot camp, when I learned why CSS is used for syling. That got me thinking; what would it been like to be a web developer in the 1990's? Would I have even been able to make a good website?

     Enter RetroBlog: A website that uses deprecated HTML styling to create a fully functional and, at least I think, does it in a way that is somewhat aesthetically pleasing while still being useful. Our blog intends to give monthly updates on different technologies from knowledgable people. RetroBlog is inspired by both 1990's HTML web development and the 1960's gruvbox color scheme. We use tables for formatting, and some jenky deprecated HTML attributes for styling. As an added challenge, you will see there are no pictures on this website.. Even our profile icons are chosen from unicode symbols. This makes the page blazingly fast and keeps things interesting.

     Improvements are to come. We aim to get at least a few knowledgable people writing for our blog while gradually improving the aesthetics over time. This is a work-in-progress, so we appreciate your patience as we make our blog more interesting and friendlier on the eyes. Stay tuned to see what's in store for the future..

     Thank you for visiting RetroBlog!!
